    Journey: Draft -> Proposer -> 2nd Line

    CAD lifecycle now exposes `PROPOSING` and `APPROVING` between draft and active outcomes.

    Spec: Document Lifecycle and Actions

  3. Status-Last Table Layout + Ownership Short Labels

    Journey: Dashboard triage

    Tables now position Status last and use RM / Proposer / Approver ownership columns.

    Spec: CAD Detail Information Architecture

  4. Inbox Scope-Centric Filtering

    Journey: Inbox triage

    Inbox center controls now filter by My Inbox and Team Inbox, with mobile rail `Me`/`Tm` pills.

    Spec: Workspace Navigation Context

  5. Home + Inbox Spacing and Filter Rhythm

    Journey: Home triage

    Search, metric grid, and filter row sequence is aligned and padding is consistent across parent/sub-filters.

    Spec: Workspace Navigation Context

  6. Portfolio Segment Exposure Cards

    Journey: Portfolio monitoring

    Portfolio cards now show Wealth, Retail, and SME Business Banking amounts, utilization, and trend minis.

    Spec: Portfolio Monitoring Hierarchy

  7. Full Form Responsiveness (CAD/CET/Sandbox)

    Journey: Group/Country review

    Nested card clipping/scroll traps are removed so values remain visible in desktop and mobile detail forms.

    Design: Layout and Responsiveness

  8. Workflow Timeline Banner

    Journey: CET initiation

    Detail banner now renders stage progression as timeline milestones with current-step emphasis.

    Design: Workflow Timeline

  9. FAB Reposition for Pagination Safety

    Journey: Quick actions

    Floating action stack sits higher to avoid overlap with table pagination in desktop and mobile screens.

    Tasks: UX Refinements

  10. Governance Administration Scope

    Journey: Policy setup and enforcement

    Governance role administers app controls and policy, without default initiation/approval rights.

    Spec: Role-Based Access Governance
